# محرك القيمة العادلة — Fair Value Engine

موقع تقييم أسهم متعدد النماذج ينفّذ منهجية «التثليث» المؤسسية على ست مراحل، للسوق السعودي (تداول، بلاحقة `.SR`) والأسواق الأمريكية.

**الموقع المباشر:** https://motl1234.github.io/fair-value-engine/

## المراحل والمعادلات

| المرحلة | المحتوى |
|---|---|
| 1. البيانات والمعايير | جلب حي من Yahoo Finance عبر وركر Cloudflare خاص (بلا مفاتيح API): أسعار سنتين + قوائم سنوية + مؤشرات. معايير Damodaran يناير 2026 (ERP سعودي 5.01٪، CRP 0.78٪، فرق تخلف Aa3 = 0.51٪) |
| 2. معدل الخصم | متوسط طريقتين: CAPM + علاوة الدولة، وعائد الدين + 3.5٪ ← WACC. بيتا الأسهم السعودية تُحسب مقابل مؤشر تاسي (بيتا Yahoo غير موثوقة للرموز السعودية) |
| 3أ. النماذج الجوهرية | DCF-FCFF (قيمة نهائية بطريقتين: جوردون + مضاعف خروج)، FCFE، توزيعات ثنائي المرحلة، الدخل المتبقي RIM (استمرارية ω=0.6)، قوة الأرباح EPV، رقم جراهم |
| 3ب. النماذج النسبية | P/E (متوسط المبرر والأقران)، EV/EBITDA، EV/Sales، P/B المبرر (ROE−g)/(ke−g)، وDCF عكسي يستخرج النمو المسعّر ضمنياً |
| 4. الدمج | ثلاث طرق تجميع (مرجح بأوزان نوع الشركة + مشذّب + وسيط)، سيناريوهات 20/60/20، هامش أمان متدرج حسب الحجم (+10٪ عند تباعد >25٪، +5٪ عند تشدد كلي)، مصفوفة حساسية WACC×g |
| 5. الطبقة الفنية | SMA50/200، RSI(14) وايلدر، MACD(12,26,9)، زخم 12−1، تقلب سنوي ← درجة مركبة [−1,+1] — توقيت لا قيمة |
| 6. التنبؤ | قصير (فني×تقلب)، متوسط (إغلاق 40٪ من الفجوة + توزيعات + زخم)، طويل (لبنات: نمو + توزيعات + رجوع تقييم) — كلها بنطاقات |

## أنواع الشركات (تُكتشف تلقائياً من القطاع)

`mature` صناعية ناضجة · `growth` نمو/تقنية · `financial` بنك/تأمين (RIM وP/B وDDM، بلا FCFF) · `dividend` توزيعات مستقرة — ولكلٍّ أوزان دمج خاصة، قابلة للتجاوز من لوحة الافتراضات.

## الوركر (worker/)

بروكسي Yahoo Finance بثلاث نقاط: `/api/summary` (quoteSummary مع crumb مُدار تلقائياً)، `/api/chart`، `/api/fundamentals` (سلاسل القوائم السنوية). النشر: `wrangler deploy` من مجلد `worker/`.

## تنويه

أداة دراسة كمية شخصية وليست توصية استثمارية. البيانات قد تتأخر 15 دقيقة، وأساسيات الشركات السعودية الصغيرة قد تنقص — راجع «سجل جودة البيانات» في أسفل كل تحليل، وعدّل الافتراضات قبل أي قرار.
